NEXT STEPS
═══════════════════════════════════════════════════════════════════
1. Clone the repository:
   git clone https://github.com/Rohit18-06/SportVerse.git

2. Setup frontend:
   cd sportsverse
   npm install
   npm run dev

3. Setup backend:
   cd backend
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   python run.py

4. Configure API keys:
   - Add CRICKET_API_KEY to backend/.env
   - Add NEXT_PUBLIC_API_URL to sportsverse/.env.local

5. Test connection:
   - Frontend: http://localhost:3000
   - Backend: http://localhost:8000/docs
